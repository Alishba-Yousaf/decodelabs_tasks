# 🤖 Rule-Based AI Assistant | DecodeLabs Internship

## Overview
This project is an entry-level implementation of a **Rule-Based Conversational Agent**. It demonstrates fundamental software engineering principles, including dictionary-based data retrieval, input sanitization, and structured control flow.

## 🛠 Technical Stack
* **Language:** Python 3.10+
* **Logic:** O(1) Dictionary Lookup Implementation
* **Design Pattern:** Command-line Interface (CLI) / Menu-Driven Architecture

## 🚀 Key Features
* **Modular Logic:** Decoupled response-mapping logic from the interaction loop.
* **Input Sanitization:** Robust handling of user input via `.strip()` and `.lower()` methods to prevent runtime errors.
* **Fallback Strategy:** Graceful handling of undefined intents, ensuring system stability.
* **User-Centric Design:** Real-time command guidance to improve user experience.

## 💻 Implementation Details
The core logic resides in a high-efficiency dictionary that maps user intents to system responses. The primary loop executes until a termination signal (`exit`) is received, maintaining session integrity.
